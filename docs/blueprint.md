# **App Name**: Trusted Vehicles Marketplace

## Core Features:

- Vehicle Listing Sync: Fetch 4-wheeler listings from an external API.
- Browse Listings: Display 4-wheeler listings in a card/grid view with a 'Verified by Trusted Vehicles Professional' badge.
- Filter & Sort: Filter and sort vehicles by price, year, kms driven, fuel type, and condition ratings.
- User Authentication: Implement user login with Google Sign-in and Phone Number + OTP verification.
- Save Vehicles: Allow logged-in users to save vehicles to Favourites or mark them as Saved for later.
- Profile Management: Enable logged-in users to manage their profile and saved listings.
- AI-Powered Condition Report Tool: Use generative AI to evaluate provided user inputs like Vehicle description, Condition & Features and Images to compose and present a consolidated condition report for the vehicle.
- AI-Powered Vehicle Recommendation Chatbot: An AI tool chatbot that can be moved around the marketplace. It has access to all vehicles and recommends vehicles based on user input (make, model, price, speed, performance, family needs, etc.). The AI adapts its tone to match the user's language (Hindi, Marathi, Urdu, English) while using English alphabets.

## Style Guidelines:

- Primary color: Deep Blue (#00008B) for trust and professionalism.
- Background color: Soft White (#F8F8FF) for a clean UI and readability.
- Accent color: Vibrant Orange (#FFA500) for energetic CTA buttons (like “Buy Now”, “Contact Dealer”).
- Body and headline font: 'Roboto' for a clean and easily readable experience.
- Use modern and easily recognizable icons for vehicle features and categories.
- Implement a card-based layout for a clean and organized display of vehicle listings.
- Incorporate subtle animations and transitions to enhance user engagement and provide visual feedback.
